### Embed Code for Carrera:
```html
<script>
  window.AI_WIDGET_TENANT = 'carrera-mistral';
  window.AI_WIDGET_LANG = 'tr';
</script>
<script src="https://yourdomain.com/widget.js" async></script>
```
